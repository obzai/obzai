# Explain, monitor and improve your vision AI systems.

**ObzAI** is a Python package for monitoring, explainability, and outlier detection in computer vision models. It provides both local and cloud-based tools for data inspection, XAI generation, anomaly detection, and visualization.

Please visit [Obz.AI](https://obz.ai/) for full offerings and documentations. 

## Features

- **Data Inspector Module**: Extracts features and detects outliers in image datasets using various statistical and deep learning-based methods.
- **XAI Module**: Generates explainability maps (e.g., CDAM, Saliency, Attention) for vision models, and provides quantitative evaluation tools.
- **Obz Client**: Enables logging and monitoring of models and explanations to the Obz cloud dashboard.
- **Tutorials**: Example notebooks for natural images, medical images, and semantic segmentation.
